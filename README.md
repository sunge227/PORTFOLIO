# 소개
대구소프트웨어고등학교 3학년 임베디드소프트웨어 개발과에 재학중인 개발자입니다. 
고등학교에 입학하여 프로그래밍을 시작했으며 그 뒤로는 학교 교육과정을 따라가며 프로그래밍 공부를 하여 공모전과 대회에 참가하였고,교내 팀 프로젝트를 진행하였습니다.

프로그래밍이 제일 재밌으나 아직 실력이 부족하여 꾸준히 공부하고 개발하고 있습니다.
